===============================
🔧 GLINAX PRIME MODEL OVERVIEW
===============================

1. ✅ User (Django Built-in)
-----------------------------
• Handles authentication and session.
• Linked to all user-specific resources.

→ Extended using `UserProfile` (OneToOne):

   - has_paid: Boolean (controls feature access)
   - payment_date: DateTime (tracks last payment)
   - paystack_customer_id: CharField (future use for recurring)
   - profile_image: ImageField (optional)
   - bio: TextField (optional)
   - created_at: DateTimeField
   - updated_at: DateTimeField

---

2. 📂 File
----------
• Stores all uploaded documents (PDF, PPTX, DOCX, etc.)

Fields:
- user: ForeignKey to User
- file: FileField or Cloud path
- file_type: CharField (e.g., pdf, pptx)
- purpose: CharField (summary, tts, etc.)
- uploaded_at: DateTimeField
- status: CharField (pending, done, failed)

---

3. 🔊 Audio
-----------
• Stores all audio generated by TTS or used for transcription

Fields:
- user: ForeignKey to User
- file: ForeignKey to File (nullable if generated from text)
- audio: FileField (path to audio file)
- source_type: CharField (tts / transcription)
- created_at: DateTimeField
- language: Optional CharField
- duration: Optional IntegerField

---

4. 🧠 Transcription
--------------------
• Stores text generated from speech (audio transcription)

Fields:
- user: ForeignKey to User
- audio: ForeignKey to Audio
- transcription: TextField
- language: Optional CharField
- created_at: DateTimeField

---

5. 💬 ChatHistory
------------------
• Logs conversations between user and AI

Fields:
- user: ForeignKey to User
- prompt: TextField (user input)
- response: TextField (AI response)
- timestamp: DateTimeField
- context: CharField (e.g., chat, summary, audio gen)
- file: ForeignKey to File (nullable)

---

6. 💭 Chat
----------
• Represents a full conversation/thread with title

Fields:
- user: ForeignKey to User (owner)
- title: CharField (suggested by AI or custom)
- created_at: DateTimeField
- is_deleted: BooleanField

---

7. 👥 ChatCollaborator
------------------------
• Allows shared access to chats between users

Fields:
- chat: ForeignKey to Chat
- collaborator: ForeignKey to User (has access)
- added_by: ForeignKey to User (who invited them)
- can_edit: BooleanField
- added_at: DateTimeField

---

8. ❓ GeneratedQuestion
------------------------
• Stores questions created by the AI from files/transcripts

Fields:
- user: ForeignKey to User
- question: TextField
- answer: TextField (optional)
- source_type: CharField (file / transcript)
- source_id: Integer (used for cross-reference)
- created_at: DateTimeField
- category: Optional CharField
- difficulty: Optional CharField

---

9. 📈 UsageLog (Optional)
--------------------------
• Records feature usage for analytics or freemium control

Fields:
- user: ForeignKey to User
- action: CharField (chat, upload, audio_gen, etc.)
- related_object: GenericRelation (optional)
- success: BooleanField
- timestamp: DateTimeField

---

10. 💳 UserPayment
-------------------
• Tracks payment transactions per user

Fields:
- user: ForeignKey to User
- paystack_reference: CharField (unique)
- amount: DecimalField
- status: CharField (pending, success, failed)
- created_at: DateTimeField
- completed_at: Optional DateTimeField
- raw_response: JSONField (optional for webhook data)

---

📌 Relationships Overview

User → UserProfile  
User → File → Audio  
User → Chat → ChatHistory  
Chat → ChatCollaborator  
Audio → Transcription  
File/Transcription → GeneratedQuestion  
User → UserPayment  
User → UsageLog (optional)
